# ProgressBar

A linear progress indicator for displaying completion percentage. Supports multiple colors and sizes.

---

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `progress` | `number` | Required | Progress value from 0 to 1 (values > 1 are capped at 1) |
| `color` | `Color` | Required | Bar fill color: `"highlight"`, `"negative"`, `"warning"`, etc. |
| `size` | `Size` | Required | Bar height: `"xsmall"` (4px), `"small"` (8px), `"medium"` (12px), `"large"` (16px), `"xlarge"` (20px) |
| `jss` | `JSS` | - | Custom styles for the container |

## With percentage label

```js
function UploadProgress({ progress }) {
  const percentage = Math.round(progress * 100);

  return (
    <Column gap="small">
      <Row justify="space-between">
        <Span>Uploading...</Span>
        <Span color="secondary">{percentage}%</Span>
      </Row>
      <ProgressBar
        color={progress === 1 ? "positive" : "highlight"}
        progress={progress}
        size="medium"
      />
    </Column>
  );
}
```

## Dynamic color based on value

```js
function StorageIndicator({ used, total }) {
  const progress = used / total;

  // Change color based on usage
  const color = progress > 0.9
    ? "negative"
    : progress > 0.7
    ? "warning"
    : "highlight";

  return (
    <Column gap="small">
      <Row justify="space-between">
        <Span>Storage</Span>
        <Span color="secondary">{Math.round(progress * 100)}% used</Span>
      </Row>
      <ProgressBar color={color} progress={progress} size="small" />
    </Column>
  );
}
```

## Animated progress

The progress bar uses CSS transform for smooth animations:

```js
function AnimatedProgress() {
  const [progress, setProgress] = useState(0);

  useEffect(() => {
    const interval = setInterval(() => {
      setProgress((p) => (p >= 1 ? 0 : p + 0.1));
    }, 500);
    return () => clearInterval(interval);
  }, []);

  return <ProgressBar color="highlight" progress={progress} size="medium" />;
}
```
