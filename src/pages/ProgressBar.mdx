# ProgressBar

The `ProgressBar` component can be used to represent percentual progress in a linear way:

<Column gap="medium">
  <ProgressBar color="highlight" progress={0.66} size="large" />
  <ProgressBar color="negative" progress={0.66} size="medium" />
  <ProgressBar color="warning" progress={0.66} size="small" />
</Column>

```js
<Column gap="medium">
  <ProgressBar color="highlight" progress={0.66} size="large" />
  <ProgressBar color="negative" progress={0.66} size="medium" />
  <ProgressBar color="warning" progress={0.66} size="small" />
</Column>
```
