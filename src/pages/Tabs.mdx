# Tabs

The `Tabs` component is made of a horizontal container with the role `tablist`, which contains tabs. It can be controlled with the keyboard and it needs to be passed an ID for the content it controls.

## Label pre renderers

You can customize how tab labels are rendered by providing a `labelRenderer` function that takes the label value and returns a ReactNode.

<Column>
  <Tabs
    aria-controls="foo"
    labelRenderer={(value) => value.split(" ").map((word, index) => index % 2 === 0 ? <strong key={index}>{word} </strong> : word + " ")}
    tabs={[
      {
        href: "/Tabs",
        selected: true,
        label: "Bold Tab 1",
      },
      {
        href: "/Tabs",
        selected: false,
        label: "Bold Tab 2",
      },
      {
        href: "/Tabs",
        selected: false,
        label: "Bold Tab 3",
      },
    ]}
  />
</Column>

## Custom tab styles

You can provide custom styles to individual tabs using the `tabJSStyle` prop, which can either be a style object or a function that receives the tab's selected state.

<Column>
  <Tabs
    aria-controls="foo"
    tabJSStyle={(data) => ({
      borderBottom: data.selected ? "2px solid cyan" : "2px solid magenta",
    })}
    tabs={[
      {
        href: "/Tabs",
        selected: true,
        label: "Styled Tab 1",
      },
      {
        href: "/Tabs",
        selected: false,
        label: "Styled Tab 2",
      },
      {
        href: "/Tabs",
        selected: false,
        label: "Styled Tab 3",
      },
    ]}
  />
</Column>
