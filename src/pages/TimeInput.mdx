# TimeInput

A styled time picker component that wraps the native HTML `<input type="time">` element. Returns hours and minutes as a Date object.

---

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `time` | `Date` | Required | Current time value |
| `onTimeChange` | `(date: Date) => void` | Required | Called when time changes |
| `bare` | `boolean` | `false` | Remove border and background styling |
| `disabled` | `boolean` | - | Disables the input |
| `addOn` | `ReactNode` | - | Icon or element to display |
| `addOnPosition` | `"start" \| "end"` | `"start"` | Position of the add-on |
| `gap` | `Gap` | `"none"` | Space between elements |
| `padding` | `Padding` | `"small"` | Container padding |

---

## Basic usage

<Column gap="medium" jss={{ maxWidth: 150 }}>
  <TimeInput time={new Date()} onTimeChange={() => {}} />
</Column>

```js
const [time, setTime] = useState(new Date());

<TimeInput time={time} onTimeChange={setTime} />
```

## Working with Date objects

The component works with JavaScript Date objects. Only the hours and minutes are used:

```js
function ScheduleForm() {
  const [startTime, setStartTime] = useState(new Date());
  const [endTime, setEndTime] = useState(new Date());

  // Get just the time portion
  const formatTime = (date) => {
    return date.toLocaleTimeString("en-US", {
      hour: "2-digit",
      minute: "2-digit",
    });
  };

  return (
    <Column gap="medium">
      <Row gap="medium" align="center">
        <Span>From:</Span>
        <TimeInput time={startTime} onTimeChange={setStartTime} />
      </Row>
      <Row gap="medium" align="center">
        <Span>To:</Span>
        <TimeInput time={endTime} onTimeChange={setEndTime} />
      </Row>
      <Span color="secondary">
        Selected: {formatTime(startTime)} - {formatTime(endTime)}
      </Span>
    </Column>
  );
}
```

## Setting initial time

```js
// Current time
const [time, setTime] = useState(new Date());

// Specific time (e.g., 9:30 AM)
const [time, setTime] = useState(() => {
  const date = new Date();
  date.setHours(9, 30, 0, 0);
  return date;
});
```
