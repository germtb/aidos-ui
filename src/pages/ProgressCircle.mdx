# ProgressCircle

The `ProgressCircle` component can be used to represent percentual progress in a circular way:

<Row gap="medium" align="center">
  <ProgressCircle progress={0.1} outerRadius={12} />
  <ProgressCircle progress={0.2} outerRadius={12} />
  <ProgressCircle progress={0.3} outerRadius={12} />
  <ProgressCircle progress={0.4} outerRadius={12} />
  <ProgressCircle progress={0.5} outerRadius={12} />
  <ProgressCircle progress={0.6} outerRadius={12} />
  <ProgressCircle progress={0.7} outerRadius={12} />
  <ProgressCircle progress={0.8} outerRadius={12} />
  <ProgressCircle progress={0.9} outerRadius={12} />
  <ProgressCircle progress={1} outerRadius={12} />
</Row>
