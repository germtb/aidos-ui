# ProgressCircle

A circular progress indicator for displaying completion percentage. Uses SVG for crisp rendering at any size.

---

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `progress` | `number` | Required | Progress value from 0 to 1 |
| `outerRadius` | `number` | Required | Radius of the circle in pixels |
| `stroke` | `number` | `4` | Width of the progress stroke |

---

## Basic usage

<Row gap="medium" align="center" padding="medium">
  <ProgressCircle progress={0.75} outerRadius={24} />
</Row>

## With label

Combine with text to show the percentage:

```js
function CircularProgress({ value }) {
  return (
    <Row align="center" gap="small">
      <ProgressCircle progress={value} outerRadius={16} />
      <Span>{Math.round(value * 100)}%</Span>
    </Row>
  );
}
```

## Skill indicator

```js
function SkillBadge({ name, level }) {
  return (
    <Row align="center" gap="medium" padding="small">
      <ProgressCircle progress={level / 100} outerRadius={16} />
      <Column gap="none">
        <Span bold>{name}</Span>
        <Span size="small" color="secondary">{level}%</Span>
      </Column>
    </Row>
  );
}

// Usage
<SkillBadge name="React" level={90} />
<SkillBadge name="TypeScript" level={85} />
<SkillBadge name="Node.js" level={70} />
```

## Animated loading

```js
function LoadingCircle() {
  const [progress, setProgress] = useState(0);

  useEffect(() => {
    const interval = setInterval(() => {
      setProgress((p) => (p >= 1 ? 0 : p + 0.05));
    }, 100);
    return () => clearInterval(interval);
  }, []);

  return <ProgressCircle progress={progress} outerRadius={20} />;
}
```

## Styling

The ProgressCircle uses these CSS variables:
- `--secondary-background`: Circle background color
- `--highlight`: Progress stroke color

The SVG is rotated -90 degrees so progress starts from the top.

---

## Use cases

- **Compact progress indicators**: Where horizontal space is limited
- **Skill/proficiency displays**: Show expertise levels
- **Loading states**: Determinate circular spinners
- **Dashboard metrics**: Compact data visualization
- **Achievement badges**: Show completion status
