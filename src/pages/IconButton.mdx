# IconButton

A circular button containing only an icon. Use IconButton for compact actions where space is limited or when the icon alone clearly communicates the action's purpose.

---

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `icon` | `IconType` | Yes | - | The icon to display |
| `size` | `Size` | Yes | - | Button size: `"xsmall"` (20px), `"small"` (24px), `"medium"` (32px), `"large"` (40px), `"xlarge"` (48px) |
| `color` | `InteractableColor` | Yes | - | `"positive"`, `"secondary"`, `"negative"`, `"dark"`, `"light"` |
| `onClick` | `function` | Yes | - | Click handler |
| `bare` | `boolean` | No | `false` | Removes background for a lighter appearance |
| `border` | `boolean` | No | `false` | Adds a border |
| `disabled` | `boolean` | No | `false` | Disables interaction |
| `aria-label` | `string` | No | - | Accessible label (recommended) |

---

## Basic variants

IconButtons come in different colors to indicate their purpose. Always pair with a clear, recognizable icon.

<Row gap="large" align="center" padding="large" justify="center">

  <IconButton size="medium" icon="fa-pencil" color="secondary" onClick={() => {}} />
  <IconButton size="medium" icon="fa-trash" color="negative" onClick={() => {}} />
  <IconButton size="medium" icon="fa-check" color="positive" disabled onClick={() => {}} />
</Row>

```js

  icon="fa-check"
  size="medium"
  color="positive"
  aria-label="Confirm"
  onClick={() => {}}
/>

<IconButton
  icon="fa-pencil"
  size="medium"
  color="secondary"
  aria-label="Edit"
  onClick={() => {}}
/>

<IconButton
  icon="fa-trash"
  size="medium"
  color="negative"
  aria-label="Delete"
  onClick={() => {}}
/>
```

## Bare variant

The `bare` variant removes the background, creating a subtle button that works well in toolbars or alongside other content.

<Row gap="large" align="center" padding="large" justify="center">
  <IconButton bare size="medium" icon="fa-check" color="positive" onClick={() => {}} />
  <IconButton bare size="medium" icon="fa-pencil" color="secondary" onClick={() => {}} />
  <IconButton bare size="medium" icon="fa-trash" color="negative" onClick={() => {}} />
  <IconButton bare size="medium" icon="fa-check" color="positive" disabled onClick={() => {}} />
</Row>

```js
<IconButton
  bare
  icon="fa-close"
  size="medium"
  color="secondary"
  aria-label="Close"
  onClick={() => {}}
/>
```

## Sizes

IconButtons scale proportionally, maintaining their circular shape at all sizes.

<Row gap="large" align="center" padding="large" justify="center">
  <IconButton size="xsmall" icon="fa-star" color="positive" onClick={() => {}} />
  <IconButton size="small" icon="fa-star" color="positive" onClick={() => {}} />
  <IconButton size="medium" icon="fa-star" color="positive" onClick={() => {}} />
  <IconButton size="large" icon="fa-star" color="positive" onClick={() => {}} />
  <IconButton size="xlarge" icon="fa-star" color="positive" onClick={() => {}} />
</Row>

```js
<IconButton size="xsmall" icon="fa-star" color="positive" onClick={() => {}} /> // 20px
<IconButton size="small" icon="fa-star" color="positive" onClick={() => {}} />  // 24px
<IconButton size="medium" icon="fa-star" color="positive" onClick={() => {}} /> // 32px
<IconButton size="large" icon="fa-star" color="positive" onClick={() => {}} />  // 40px
<IconButton size="xlarge" icon="fa-star" color="positive" onClick={() => {}} /> // 48px
```

## Accessibility

Since IconButtons have no visible text, always provide an `aria-label` to describe the action for screen reader users.

```js
// Good - has accessible label
<IconButton
  icon="fa-trash"
  size="medium"
  color="negative"
  aria-label="Delete item"
  onClick={() => {}}
/>

// Bad - no accessible label
<IconButton
  icon="fa-trash"
  size="medium"
  color="negative"
  onClick={() => {}}
/>
```

Consider using a `Tooltip` component alongside IconButton to provide visual hints for all users.
