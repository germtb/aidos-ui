# Icon

The `Icon` component displays vector icons using [Iconify](https://iconify.design/) under the hood, giving you access to 200,000+ icons from 150+ icon sets including Font Awesome, Material Design, and more.

---

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `icon` | `IconType` | Yes | The icon identifier (e.g., `"fa-check"`, `"fa-github"`) |
| `size` | `Size` | Yes | Icon size: `"xsmall"` (8px), `"small"` (12px), `"medium"` (18px), `"large"` (24px), `"xlarge"` (32px), `"xxlarge"` (48px), `"xxxlarge"` (64px) |
| `color` | `TextColor` | Yes | Icon color: `"primary"`, `"secondary"`, `"highlight"`, `"negative"`, `"subtle"`, `"light"` |
| `ariaLabel` | `string` | No | Accessible label for screen readers |

## Accessibility

Always provide an `ariaLabel` when the icon conveys meaning that isn't available through surrounding text.

```js
// Decorative icon (no label needed if text explains it)
<Button>
  <Icon icon="fa-download" color="light" size="small" />
  Download
</Button>

// Meaningful icon (needs label for screen readers)
<Icon
  icon="fa-warning"
  color="negative"
  size="medium"
  ariaLabel="Warning"
/>
```

## Preloading icons

By default, Iconify loads icons asynchronously from a CDN. To prevent flickering on initial render, you can preload icons by bundling them with your application.

**Option 1: Preload an entire icon set**

```bash
npm install @iconify-json/fa
```

```js
import { addCollection } from "@iconify/react";
import faIcons from "@iconify-json/fa/icons.json";

// Call once at app initialization (e.g., in _app.tsx)
addCollection(faIcons);
```

**Option 2: Preload individual icons**

```bash
npm install @iconify-icons/fa
```

```js
import { addIcon } from "@iconify/react";
import faCheck from "@iconify-icons/fa/check";
import faClose from "@iconify-icons/fa/close";

addIcon("fa:check", faCheck);
addIcon("fa:close", faClose);
```

## Finding icons

Browse available icons at [Iconify Icon Sets](https://icon-sets.iconify.design/). The component uses Font Awesome 4 icons by default (prefixed with `fa-`), but you can use any Iconify-supported icon set.
